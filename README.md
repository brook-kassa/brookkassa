# Brook Kassa Portfolio

A tactile personal brand website built with React, Vite, Tailwind CSS, and Framer Motion.

## Stack

- React + Vite
- Tailwind CSS via `@tailwindcss/vite`
- Framer Motion
- React Icons

## Highlights

- Neumorphic, layered interface with soft depth and dual-light shading
- Hero section with subtle parallax and animated gradient response
- Social hub that replaces a generic link page
- Photo gallery with modal viewing
- Video highlight section ready for YouTube expansion
- Identity and future-growth sections for blog, projects, and services
- Dark/light theme toggle adapted to the neumorphic surface system
- Local SVG media placeholders that can be swapped for real assets later

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deployment

The repository includes an Azure Static Web Apps workflow. The app builds to `dist`.
